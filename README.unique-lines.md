`unique-lines.cmd`: Lets you see the unique lines a file has, compared to another file.  
Syntax: `unique-lines.cmd a.txt b.txt`  
 - `a.txt`: The file you want to see unique lines of.
 - `b.txt`: The file containing lines.

Example:
 - `a.txt`:
```
1
4
2.2455
cfdfdfd
9
```
 - `b.txt`:
```
3
9
4
12.45
1
```

Result:
```
-- a.txt
2.2455
cfdfdfd
```
